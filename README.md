Purpose: This repo contains reproducible and/or adaptable EEG analysis pipelines implemented in Python and R
Piplelines stages: Data import, preprocessing (filteirng, re-ref), Artifact correction (ICA/rejection), extraction (segmenting, ERP, spectral power, time-frequency etc) Statistical analysis 
Tools: Python (MNE-Python, NumPy, SciPy, pandas) R (tidyverse etc)
All scripts are designed to be adapted and reproducible
